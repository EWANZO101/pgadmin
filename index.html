Welcome to the SnailyCAD Helper Site

This site offers a comprehensive set of tools designed to help you manage your SnailyCAD server effortlessly. Whether it's resetting passwords, disabling authentication methods, or managing databases, we’ve got you covered.

Web-based Server Management Tool

This web-based tool empowers administrators to manage their servers, reset passwords, disable authentication methods, and perform various server management tasks remotely.

  • Password Reset
  • Force Disable Discord Authentication
  • Easy Access to Database Management

Important Information

Ensure that PostgreSQL port 5432 is open and remotely accessible for smooth communication between the app and your database.

Please note that after sensitive operations (e.g., password reset or authentication changes), all relevant data will be wiped for security purposes.

Need Assistance?

If you’re facing difficulties with port forwarding, server setup, or anything else, we’re here to help! Connect with us through Discord or via email.
